Данный HTML-документ является основной страницей веб-калькулятора, стилизованного под бренд языка Rust — с логотипом, полем ввода и полным набором кнопок для вычислений. Разметка построена модульно: отдельные файлы CSS и JavaScript отвечают за дизайн и логику, что упрощает поддержку и масштабирование проекта. Страница обеспечивает адаптивный интерфейс и готовую основу для реализации полноценного инструмента расчётов.
